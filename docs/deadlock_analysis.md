# 교착(deadlock) 분석 — 전 모델 × 전 레짐

IL 정책의 실패를 "교착(막혀서 멈춤)"과 "느림(진전 중 timeout)"으로 분리해 7개 모델을
소형 개방맵부터 대형 벽맵까지 한 번에 측정한 기록. 성공률만으로는 안 보이는 **실패의 성격**을 본다.

- 데이터: `outputs/deadlock_small.json`(empty-8-8/16-16), `outputs/deadlock_big.json`(random/room/maze)
- 재현: `py scripts/eval_deadlock.py --bench-dir <bench> --maps <...> --agents <...> --ckpts <7종> --out <name>.json`
- MovingAI 25 인스턴스/셀, CBS 없음(IL 롤아웃만) → 빠름.
- 그림 재현: `py scripts/plot_report_figures.py --deadlock` → `docs/img/fig_deadlock_all.png`

![deadlock all](img/fig_deadlock_all.png)

## 지표 정의
- **succ** = 전원 충돌 없이 목표 도달 비율.
- **reached** = 에이전트 중 목표 도달 평균 비율(부분 점수).
- **DL** = 전체 중 "교착" 비율. 교착 = 실패 에피소드에서 마지막 12스텝간 잔여 맨해튼거리 진전 0(얼어붙음/진동).
- **DL/fail** = *실패한 것들 중* 교착 비율. **이게 실패의 성격을 가른다** — 1.0이면 "실패=전부 막힘", 낮으면 "느리거나 길을 잃은 실패"도 섞임.
- **rem** = 종료 시 평균 잔여거리(작을수록 목표 근처까지 감).

모델 7종: `MLP-BC`, `CNN-BC(1.6k)`, `CNN-BC(13.7k=diverse)`, `CNN-big(27.7k)`, `CNN-bigdense(47k)`, `CNN-DAgger`, `CNN-DAgger-ext`. (CNN들은 best-val 재학습본)

## 발견

1. **IL 실패 = 거의 곧 교착.** 개방맵(empty)에서 모든 CNN의 **DL/fail ≈ 1.00** — 실패하면 timeout이 아니라
   막혀서 멈춘다. 즉 "IL의 성공률을 올린다 = 교착을 줄인다"가 거의 등식. 성공률만 보던 걸 교착률로 봐도
   결론이 같은 이유.

2. **DAgger의 작동 방식이 지표로 잡힌다.** `DAgger-ext`는 개방·저밀도 대형맵에서 성공률을 올릴 뿐 아니라
   **DL/fail을 1.0 → 0.6~0.75로 낮춘다**(empty-8-8 n4, random-10 n4·n8). 곧 "막힘 실패"를 "느리지만 진전 중
   실패"로 바꾼다 — 교착 탈출을 학습했다는 직접 증거. 성공까진 못 가도 실패의 질이 달라짐.

3. **작은 풀 DAgger는 대형맵을 오히려 망친다.** `CNN-DAgger`(소형 개방 풀)는 random-32-32-10 n4에서
   0.52(big)→**0.32**로 후퇴. 풀을 넓힌 `DAgger-ext`가 이걸 0.68로 되살림. → DAgger는 **학습 분포가 닿는 곳에서만**
   교착을 푼다(작업 로그 결론 재확인, 이제 대형맵 수치까지 포함).

4. **데이터 규모가 교착을 단조 감소시킨다.** empty-8-8 n4 DL: MLP .88 → BC1.6k .72 → 13.7k .44 → big .36.
   best-val 재학습 후 `CNN-big`이 소형 포화맵에서 `diverse`를 앞섬(succ 0.56→0.64).

5. **밀도·벽이 늘면 실패의 성격이 "교착"에서 "길 잃음/느림"으로 이동.** random-20·room·maze에서 DL/fail이
   0.44~0.76으로 떨어짐 — 잔여거리(rem)가 큰 채 끝나는(목표 근처도 못 감) 실패가 늘어난다. 5×5 FOV로는
   벽 미로에서 전역 경로를 못 잡는 구조적 한계. 교착 회복(DAgger)으로 풀 문제가 아님.

6. **미해결 벽:** room/maze는 7모델 전부 succ≈0.00, reached도 0.01~0.34. 흥미롭게 **`CNN-big`이 maze에서 DL을
   가장 낮게**(.44~.48) 유지 — 큰맵 학습이 "막히진 않게" 하지만 목표까지 잇지는 못함. 이 구간은 교착이 아니라
   **전역 경로 부재**가 병목 → FOV 확장/전역 힌트(경로 채널) 같은 표현 변경이 필요.

## 한 줄 요약 (모델별 교착 시그니처)
- **MLP**: 전 구간 붕괴, 실패=순수 교착. 주력 제외.
- **CNN-BC 계열**: 데이터 늘수록 교착↓. `big`(best-val)이 소형 포화·저밀도 대형에서 최상위 BC.
- **bigdense**: 중밀도 대형(random-10)에서 교착 완화에 기여, 소형 포화엔 무효.
- **DAgger**: 소형 개방맵 교착 특효, 단 대형맵엔 소형 풀이면 독.
- **DAgger-ext**: **개방·저밀도 전 구간 최강**, 교착을 진전-실패로 전환. 고밀도·벽맵엔 무력(분포 밖).

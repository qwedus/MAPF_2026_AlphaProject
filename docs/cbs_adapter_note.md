# CBS Adapter Note

This branch prepares the repository structure for integrating an open-source
Conflict-Based Search (CBS) implementation.

## Current Scope

- Add source, script, documentation, third-party, and output directories.
- Define a minimal adapter interface in `src/cbs_adapter.py`.
- Provide a placeholder example runner in `scripts/run_atb033_example.py`.

## Not Included Yet

- CBS algorithm implementation.
- Third-party solver import.
- ATB033 scenario parsing.
- Path output generation.
